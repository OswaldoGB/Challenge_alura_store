# Challenge_alura_store

Análisis geográfico de ventas

El análisis de las coordenadas geográficas permitió visualizar la distribución espacial de las ventas realizadas por cada tienda. A través de gráficos de dispersión y mapas de calor se identificaron áreas con mayor concentración de compras, lo que indica regiones donde la demanda de productos es más alta.

Los resultados muestran que ciertas zonas presentan una mayor densidad de ventas, lo que puede estar asociado a factores como densidad poblacional, accesibilidad o preferencias regionales de los consumidores. Este tipo de análisis permite comprender mejor el comportamiento geográfico del mercado y puede ser útil para definir estrategias comerciales o logísticas en el futuro.
